Application scripts

Standalone JavaScript Modules

As a convenience, we provide the entire UI framework broken down in separate modules. Some of the standalone modules were not used with every theme but ALL of the modules are 100% compatible with any of the provided themes.

  • module-essentials.js - the required JavaScript for Essential UI components such as Tabs and Tree Views
  • module-layout.js - the required JavaScript for Layout components
  • module-maps.js - the required JavaScript for Maps components (Vector Maps & Google Maps)
  • module-media.js - the required JavaScript for Media components, such as Carousels, Galleries and Media Players
  • module-navbar.js - the required JavaScript for Navbar components
  • module-sidebar.js - the required JavaScript for Sidebar components

All Standalone JavaScript Modules Bundle

Each theme includes an extra Standalone JavaScript Modules Bundle that contains ALL THE AVAILABLE standalone JavaScript modules. The bundle file name is module-bundle-all.js. There is also a minified version module-bundle-all.min.js within the same directory.

Theme-specific Standalone JavaScript Modules Bundle

Each theme includes a Theme-specific Standalone JavaScript Modules Bundle that contains only the JavaScript modules used for the theme as showcased in the demo. The bundle file name is module-bundle-main.js. There is also a minified version module-bundle-main.min.js within the same directory. This is the default file loaded in every theme and it's the recommended way, even more so if you're using the grunt watchers included with the development suite.

Vendor scripts

Vendor Script Bundles

As a convenience, we provide separate 3rd party script bundles used for the entire UI framework. Some of the bundles were not used with every theme but ALL of the bundles are 100% compatible with any of the provided themes.

  • vendor-core.js

    Includes, in this exact order:

    1. jQuery
    2. Bootstrap
    3. jQuery Breakpoints
    4. jQuery Nicescroll
    5. Grid-A-Licious
    6. jQuery Cookie
    7. Holder.js
    8. Handlebars
    9. A custom build of jQuery UI with only the following components:
      • Core
      • Widget
      • Mouse
      • Position
      • Draggable
      • Droppable
      • Resizable
      • Selectable
      • Sortable
    10. jQuery Hotkeys

  • vendor-tables.js

    Includes, in this exact order:

    1. jQuery Datatables
    2. Datatables Bootstrap Integration Plugin

  • vendor-forms.js

    Includes, in this exact order:

    1. Seiyria Bootstrap Slider
    2. Bootstrap Select Plugin
    3. Bootstrap Switch
    4. Bootstrap3 Datepicker
    5. jQuery Minicolors

  • vendor-media.js

    Includes, in this exact order:

    1. jQuery jPlayer
    2. jPlayer Playlist Plugin
    3. Owl Carousel

  • vendor-tree.js

    Includes, in this exact order:

    1. jQuery FancyTree Plugin

  • vendor-charts.js

    Includes, in this exact order:

    1. Flot Charts
    2. jQuery Sparkline
    3. Easy Pie Chart
    4. Raphael (required by Morris)
    5. Morris.js

  • vendor-maps.js

    Includes, in this exact order:

    1. jQuery Vector Maps

All Vendor Scripts Bundle

Each theme includes a Vendor Scripts Bundle with ALL THE AVAILABLE 3rd party JavaScript libraries. The vendor bundle file name is vendor-bundle-all.js. There is also a minified version vendor-bundle-all.min.js within the same directory.

  • Includes, in this exact order:

    1. vendor-core.js
    2. vendor-tables.js
    3. vendor-forms.js
    4. vendor-media.js
    5. vendor-charts.js
    6. vendor-maps.js
    7. vendor-tree.js
PRO TIP: You can get the full list of all the various libraries installed and the versions used and also check for available updates for each library in seconds, anytime! Just go into a Terminal from the root directory and type bower list. After updating a package using Bower, you only need to run grunt --theme your_theme_name build:dist to recreate all the bundles with the updated plugins.
